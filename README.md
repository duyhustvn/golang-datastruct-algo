# golang-datastruct-algorithm

This repo hosts code for datastruct and algorithm in golang. 

# Benchmark
- Go to folder that has benchmark file
- Run command
```
go test -bench=. -count=5 -run=^#
```
 - -count=5 run 5 times
 - -run=^# not running any test case, only run benchmark

# Test
- Run command

```
go test ./dp/fib
```
